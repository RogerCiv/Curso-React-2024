# 06-CRUD-Firebase Project

## Overview

This project is a CRUD application utilizing Firebase as the backend service. It allows users to create, read, update, and delete data stored in Firebase.

## Features

- **Create**: Add new entries to the database.
- **Read**: Retrieve and display data from Firebase.
- **Update**: Modify existing data entries.
- **Delete**: Remove data entries from the database.

## Technologies Used

- **Firebase**: For the backend database and hosting services.
- **React**: To build the user interface.
- **React Router DOM**: For routing and navigation in the app.
- **SweetAlert2**: To display alert messages.
- **Tailwind CSS**: For styling the application.

## Installation

To get started with this project, clone the repository and install the dependencies:

```bash
git clone <repository-url>
cd 06-crud-firebase
pnpm install
```
